Discrete Structure: Magic Square Games (2022-2023)

This project is an interactive game based on the concept of a magic square, developed as part of my Discrete Structure course. A magic square is a grid where the numbers in each row, column, and both diagonals all add up to the same total. The game allows users to arrange numbers and test whether their grid forms a valid magic square. It was built using HTML, CSS, and JavaScript, with a focus on logic implementation and user interaction.
